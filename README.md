### 🚀 BeeconMini Seed AC1 原生 OpenWrt 固件云编译

本项目基于 GitHub Action工作流，你无需配置本地复杂的编译环境，就能一键自动化构建出属于你自己的 BeeconMini Seed AC1 原生 OpenWrt 固件。

#### 🛠️ 使用方法：

- 按照你自己的实际需求修改配置文件。

- 触发 GitHub Actions 自动构建。

- 构建完成后，前往项目的 发布页 (Releases) 下载编译好的固件刷入设备。

#### 💡 注意事项：

固件的底层适配代码来自开源项目，本人仅搬运并搭建了自动构建工作流，因此不对适配引入的 Bug 负责。不过别担心，经过我一段时间的深度使用，目前稳定性良好，暂未遇到任何问题。
